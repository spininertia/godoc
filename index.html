Godoc

Minimal google-doc like code editor.

Credit

This project relies on many excellent open source projects, here is the list:

  • express, web app framework in nodejs.
  • bootstrap, css tooklit.
  • ot, nodejs operational transformation library.
  • ot-demo, prototype front end(centralized). Godoc-forntend extends it.
  • socket.io, websocket for nodejs.
  • code mirror, text editor component.
  • jQuery
  • bower
